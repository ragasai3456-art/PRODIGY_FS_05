# 📱 Social Media Platform
A full-stack social media application where users can create accounts, share posts, like, and comment on posts.
## 🚀 Features
- User Registration & Login (JWT Authentication)
- Create Posts
- Like Posts ❤️
- Comment on Posts 💬
- View Feed
- Follow Users 👥
## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: JWT
